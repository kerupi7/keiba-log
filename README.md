# keiba-log

JRA中央競馬の予想を発走前に公開し、レース後に結果と答え合わせを追記する個人運営の記録サイトです。

- 予想は発走前のオッズ・分析に基づく主観的な評価であり、的中を保証するものではありません。
- 馬券の購入は自己責任で行ってください。当サイトは購入を推奨するものではありません。
- 外れた予想も含め、すべての結果をそのまま公開する方針です（回収率・的中率で長期的に評価してもらうため）。

## サイト構成

- `index.html` — 通算成績サマリ＋レース一覧
- `race.html?id={race_id}` — レース詳細（予想全文＋結果）
- `stats.html` — 通算成績の詳細（券種別ROI・KPI・校正）

データは `data/` 配下のJSONをクライアント側JSが直接fetchして描画しています（ビルド工程なし）。
